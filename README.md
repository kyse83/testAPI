# testAPI

Support numérique interactif pour la brasse en piscine.

## Contenu
- `index.html` : visualisation 3D animée de la nage de brasse avec étapes guidées (GSAP, Three.js, CreateJS).

## Utilisation
1. Ouvrez `index.html` dans un navigateur moderne (Chrome, Firefox, Edge).
2. Cliquez sur **Mouvement suivant** ou sur les pastilles pour parcourir les phases de la brasse.
