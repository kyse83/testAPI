# testAPI

Visualisation interactive d'un homme réaliste en 3D avec vue aérienne et contrôles orbitaux.

## Contenu
- `index.html` : scène Three.js chargée avec un modèle animé glTF (Soldier) et commandes pour la vue top-down ou l'auto-rotation.

## Utilisation
1. Ouvrez `index.html` dans un navigateur moderne (Chrome, Firefox, Edge, Safari).
2. Faites glisser pour orbiter, molette pour zoomer, clic droit pour translater.
3. Utilisez **Recentrer & vue top** pour replacer la caméra au-dessus du modèle, ou **Auto-rotation** pour un balayage continu.
